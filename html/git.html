Git

  • git init - инициализирует новый репозиторий Git в текущей папке. Создает скрытую директорию .git.
  • git add . - добавляет все измененные файлы в текущей директории и ее поддиректориях в область подготовленных файлов (staging area).
  • git commit -m "сообщение" - создает новый коммит (сохранение) с所有 добавленными через git add файлами и комментарием, описывающим изменения.
  • git status - показывает текущее состояние рабочего каталога и staging area: какие файлы изменены, но не добавлены, какие добавлены, но не закоммичены.
  • git log --oneline - показывает историю коммитов в текущей ветке в сокращенном, удобном для чтения виде (хэш коммита и его сообщение).
